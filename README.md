# AI Resume Screening System

AI-powered resume screening platform built using FastAPI, React and SQLite.

## Features

- User Authentication
- Candidate Management
- Job Management
- Analytics Dashboard
- Resume Matching
- REST APIs
- SQLite Database

## Tech Stack

Backend:
- FastAPI
- SQLAlchemy
- SQLite

Frontend:
- React
- Axios
- Vite

## Run Backend

cd backend

uvicorn app.main:app --reload

## Run Frontend

cd frontend

npm install

npm run dev
